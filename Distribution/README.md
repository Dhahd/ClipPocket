# ClipPocket v2.5.0

Your Smart Clipboard Manager for macOS

## Installation

1. **Drag ClipPocket.app to the Applications folder**
   - Simply drag the ClipPocket icon to the Applications folder shortcut

2. **Launch ClipPocket**
   - Open your Applications folder and double-click ClipPocket
   - If you see a security warning, go to System Settings → Privacy & Security and click "Open Anyway"

3. **Grant Permissions**
   - ClipPocket will request Accessibility permissions to monitor clipboard
   - Click "Open System Settings" and enable ClipPocket in the Accessibility list
   - Restart ClipPocket after granting permissions

## What's New in v2.5.0

- **Configurable Show Delay** - Set delay (0-2s) before window appears when mouse hits bottom edge
- **Configurable Hide Delay** - Set delay (0-2s) before window hides when mouse moves away
- **Smart Auto-Hide** - Manual opens use fixed 2s delay for comfortable mouse travel from menu bar
- **Enhanced Edge Detection** - More reliable auto-show when enabled in settings

## System Requirements

- macOS 13.5 (Ventura) or later
- Apple Silicon or Intel Mac

## Quick Start

1. Click the clipboard icon in your menu bar or press **⌘⇧V** (Cmd+Shift+V)
2. Enable "Auto Show on Edge" in Settings to show ClipPocket by moving mouse to bottom of screen
3. Customize show/hide delays to your preference
4. Copy something - it's automatically saved!
5. Right-click items to pin your favorites

## Privacy

- All clipboard data stays on your Mac
- No cloud sync or external servers
- Use Incognito Mode to pause monitoring
- Clear history anytime from Settings
