## ğŸ“¦ Installation

### Step 1: Download
Download `ClipPocket-Installer.dmg` from the Assets section below

### Step 2: Install
1. Open the DMG file
2. Drag `ClipPocket.app` to the `Applications` folder
3. Launch ClipPocket from Applications

### Step 3: Fix Security Warning (if needed)

If you see **"ClipPocket is damaged"** or similar warning:

**Option 1: Terminal Command (Recommended)**
```bash
xattr -cr /Applications/ClipPocket.app
```

**Option 2: System Settings**
1. Open **System Settings** > **Privacy & Security**
2. Scroll down to find the ClipPocket warning
3. Click **"Open Anyway"**

This is a standard macOS security measure for apps not distributed through the App Store. ClipPocket is safe and open source!

### Step 4: Grant Permissions
Follow the onboarding wizard to grant:
- **Accessibility** permission (for keyboard shortcuts)
- **Screen Recording** permission (for clipboard monitoring)
